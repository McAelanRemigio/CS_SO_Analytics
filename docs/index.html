Project Overview

This report demonstrates a reusable survey analytics framework developed in R to transform structured and open-ended stakeholder feedback into operational insights and executive-style reporting outputs.

The workflow included:

  • Survey data cleaning and normalization
  • Categorical response aggregation
  • Text-based thematic analysis using rule-based keyword detection
  • Stakeholder segmentation and filtering
  • Automated reporting tables and visual summaries
  • Strategic insight synthesis for decision support

Technical Workflow

The analytics workflow leveraged the R ecosystem, including:

  • dplyr for transformation and wrangling
  • stringr for text parsing and thematic detection
  • ggplot2 for visual reporting outputs
  • tidyr and purrr for scalable data processing
  • knitr and Quarto for reproducible reporting
