# Program Lifecycle Management Analytics Pipeline

This project contains an anonymized analytics framework for standardizing fragmented program lifecycle management data into reusable operational reports.

The workflow was designed to consolidate multi-year exports across program activity, partner engagement, platform usage, and outcome-related metrics. The public version removes all raw data, organization-specific identifiers, and internal report outputs while preserving the reusable analysis structure.

## Project Goals

- Standardize fragmented operational exports into a consistent reporting format
- Define reusable KPI logic across program areas
- Produce monthly trend summaries and academic-year comparison tables
- Reduce manual reconciliation across inconsistent source files
- Support leadership reporting for planning, engagement, operations, and evaluation

## Public Version Notes

Raw data is excluded for privacy. Synthetic sample data may be used to demonstrate the pipeline structure. Any organization-specific names, file paths, branding, and internal figures have been removed.
